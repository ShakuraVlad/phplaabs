### Циклы

Создать файл index.php со следующим кодом:

```
<?php

$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
   $a += 10;
   $b += 5;
}

echo "End of the loop: a = $a, b = $b";
```

Добавьте вывод промежуточных значений $a и $b на каждом шаге цикла.
```
$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
   $a += 10;
   $b += 5;

   echo "Шаг $i: a = $a, b = $b <br>";
}

echo "End of the loop: a = $a, b = $b";
```
Перепишите этот цикл, используя оператор while и do-while.

```
//Через While


$a = 0;
$b = 0;
$i = 0;

while ($i <= 5) {
    $a += 10;
    $b += 5;

    echo "Шаг $i: a = $a, b = $b <br>";

    $i++;
}

echo "End of the loop: a = $a, b = $b";

//Через do-while

$a = 0;
$b = 0;
$i = 0;

do {
    $a += 10;
    $b += 5;

    echo "Шаг $i: a = $a, b = $b <br>";

    $i++;
} while ($i <= 5);

echo "End of the loop: a = $a, b = $b";
```

## Контрольные вопросы

1. Разница между циклами:

for — когда известно число повторений
while — проверка до выполнения (может не выполниться)
do-while — сначала выполняется, потом проверка (минимум 1 раз)

2. Тернарный оператор ? :

Короткий if-else:

$result = (условие) ? "если да" : "если нет";

3. do-while с ложным условием

Выполнится 1 раз, потому что проверка идёт после выполнения.
